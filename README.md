# Setup

This is a [T3 Stack](https://create.t3.gg/) that uses:
- [Next.js](https://nextjs.org)
- [NextAuth.js](https://next-auth.js.org)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)

# Dev

- `curl -fsSL https://bun.sh/install | bash`
- `bun install`
- `bun dev`
